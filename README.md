# ⚔️ Age Of Dragons

## Sobre o Projeto

Um mini-jogo de combate por turnos você controla um guerreiro e enfrenta dragões.

---

## Tecnologias Utilizadas

- HTML5 - estrutura semântica
- CSS3 - estilização
- Bootstrap - responsividade e layout
- JavaScript - lógica do jogo e consumo da API
- Git e GitHub - versionamento
- Vercel - deploy da aplicação

---

## Como Executar

1. Clone o repositório
2. Abra a pasta do projeto
3. Abra o arquivo `index.html` no navegador

Navegue para as outras página a partir da página `index.html`

## Estrutura de Arquivos

age-of-dragons/
